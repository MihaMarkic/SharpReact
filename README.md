This is an implementation of React principles to .NET.

## Requirements

.NET Standard 2.0 for Core and Router, .NET 4.6.1 for WPF.

##  State

Currently it supports WPF frontend (a very limited number of controls and their properties) but it could support other UI frameworks as well.
